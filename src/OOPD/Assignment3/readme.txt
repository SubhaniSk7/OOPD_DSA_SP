How to Run:
	>> javac *.java
	>> java MakeMyTrip

Application work flow:

Login can be done using Id.

1. Initially superAdmin will be created(Admin) and admin will monitor all the activities by login.
2. Company can be signup and then provide services accordingly.
	- company can be update BookingCharges for it, updateOfferCost which will reflect in bookingCost.
3. User can signUp and then login.
	- user can book tickets for Bus, Train, Flight.
	- user can his journey view History.
	- user can cancel journey tickets with their tripId.
