To Run program:
	1.run server.c --> $./server
	2.run client.c with argument of server ip as--> $./client <serverIP>
	
	3.After Successful connection establishment, enter message to send to Server.
	4.server will receive and reverses the string.
	5.server send the reversed string to client.
	
Note:
	We can send only one request message to server(not a loop).If we have to send another message, then re-run the programs.
